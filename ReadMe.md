# Folder Size Sorting Tool

## Introduction

This is a graphical tool built with Python and PyQt5 to view and sort the size of files and folders within a specified directory. It helps users easily view the contents of a folder and sort them in ascending or descending order by name or size.

## Features

- ğŸ–±ï¸ **Drag and Drop Support**: Allows dragging and dropping folders into the input field to automatically load folder contents.
- ğŸ”„ **File Size Sorting**: Clicking the header allows sorting by name or size in ascending or descending order.
- ğŸ“‹ **Re-sort Button**: Clicking the re-sort button sorts the folder contents by size.
- ğŸ–±ï¸ **Right-Click Menu**: Provides right-click menu options to reset the path and delete the path.

4. Right-click on the table contents to select "Reset" or "Delete" the path.
